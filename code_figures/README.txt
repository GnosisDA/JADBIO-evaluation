
#### Overall description.

This folder contains scripts for reproducing the figures presented in the paper "Just Add Data: Automated Predictive Modeling and BioSignature Discovery" bioRxiv 2020.05.04.075747 (Preprint, 2020).

#### Description of the files in the folder.

- Figure_4, Figure_5, S_Figure_1, S_Figure_2: each subfolder contain one or more R scripts for producing the corresponding figure.

- ancillaryFunction.R: an R function for extracting the legend from a ggplot.

- auto-sklearn_results.RData, JADBIO_results.RData: R binary files each containing the results of the respective system in tabular (data.frame) format.

- session_info.txt: version of R and packages used for producing the figures.
